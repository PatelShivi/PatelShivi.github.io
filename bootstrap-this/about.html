Instruction on how to use bootstrap

Open Source

Bootstrap is a open source framework. It is the world’s most popular framework for building responsive, mobile-first sites, with BootstrapCDN and a template starter page using HTML, CSS and JS.

Mobile First

Bootstrap includes mobile first flexbox grid with twelve column to build layouts.

Utilities

Bootstrap provides utilities to design your code same as css. Colors, Borders, Spacing, Clearfix, Float and more are bootstrap utilities.

Color

Using color, we can change the color of any component. It is useful for changing background color and text color as well.

Border

Border is used for give border to component. Codes are given for Color borders also.

Flex

Flex can be used to manage the layout, alignment, and sizing of grid columns, navigation, components, and more with a full suite of responsive flexbox utilities. .

Components Library

Bootstrap comes with components like cards, buttons, jumbotrons, collapse and many more. It is very to use components from bootstrap as it comes with all the instruction on how to use it.

Navbar

Navbar is used for display navigation with bootstrap. It comes with many option for navigation.

Cards

Card is one of the bootstrap content container. It is use for putting hraders and footers.

Jumbotrons

Jumbotrons is also a bootstrap component. it is use to display your content with hreaing.
